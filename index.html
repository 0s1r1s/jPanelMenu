jPanelMenu is a jQuery plugin that creates a paneled-style menu (like the type seen in the mobile versions of Facebook and Google, as well as in many native iPhone applications).

Usage

jPanelMenu is extremely easy to use. Build your page however you’d like to (the source order does not matter), and then just tell the plugin which element is your menu and which is your trigger.
